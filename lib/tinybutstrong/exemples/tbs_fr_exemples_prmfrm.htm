Exemple d'utilisation du paramètre frm

Le paramètre frm permet d'afficher une valeur date-heure ou numérique en lui appliquant un format.

 

Formats de date

Avec la date et heure actuelle
Format Affichage
yyyy-mm-dd [onshow..now;frm='yyyy-mm-dd']
dd/mm/yyyy [onshow..now;frm='dd/mm/yyyy']
hh:nn:ss [onshow..now;frm='hh:nn:ss']
dddd dd mmmm [onshow..now;frm='dddd dd mmmm']
ddd dd mmm [onshow..now;frm='ddd dd mmm']
dd-mm-yyyy hh:nn [onshow..now;frm='dd-mm-yyyy hh:nn']

Formats numériques

Avec la variable Php $amount = 1023.2568
Format Affichage
0 000,00 € [onshow.amount;frm='0 000,00 €']
0 000. € (1) [onshow.amount;frm='0 000. €']
(1) Le point (.) au bout signifie qu'il n'y a aucune décimale.

Avec la variable Php $rate = 0.751897
0,000 % [onshow.rate;frm='0,000 %']

Formats conditionnels

Avec la variable PHP $amout2 = -255.4893
Format Affichage
+0 000,00|-0 000,00|0 [onshow.amount2;frm='+0 000,00|-0 000,00|0|-']

Formats réutilisables

Les formats curr1 et rate1 sont définis grâce au paramètre 'tplfrms'.
[onload;tplfrms;curr1='0 000,00 €';rate1='0,000 %']
Format Affichage
curr1=0 000,00 €
 [onshow.amount;frm=curr1]
rate1=0,000 %
 [onshow.rate;frm=rate1] %
