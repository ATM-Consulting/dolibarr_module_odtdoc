Exemple de paramètre att

La paramètre att permet de modifier à distance les attributs de balises Xml/Html.
Cela peut être très précieux quand il n'est pas facile d'attteindre le code source du modèle (OpenOffice, Ms Office 2007, Dreamweaver, ...). Ou simplement si vous voulez coder un modèle Html qui soit compatible W3C.
