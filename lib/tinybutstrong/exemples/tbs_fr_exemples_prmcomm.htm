Exemple d'utilisation du paramètre comm

Le paramètre comm offre un confort supplémentaire pour l'élaboration des modèles avec un éditeur HTML visuel (tel que Dreamweaver ou FrontPage). Il permet d'étendre les bornes d'un indicateur aux bornes du commentaire HTML qui l'entour.

 

Avec le paramètre comm : Il est

Sans le paramètre comm : Il est [onshow..now;frm='hh:nn:ss']

Le résultat de la fusion est identique avec ou sans le paramètre 'comm'. Notez que l'indicateur avec le paramètre 'comm' n'est visible que sur le source du modèle HTML.
