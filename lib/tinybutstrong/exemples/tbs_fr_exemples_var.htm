Exemple de fusion de variable Php


Les champs automatiques ([onload] et [onshow]) permettent d'afficher une variable PHP dans votre modèle. Vous pouvez afficher une variable simple, ou alors l'item d'une variable tableau, ou encore une propriété d 'une variable objet. Il existe aussi plusieurs paramètres de champs TBS qui permettent de mettre en forme l'affichage de la valeur.


Variable Php Champ automatique
$montant [onshow.montant] €
$tache['lundi'] [onshow.tache.lundi]
$obj->param [onshow.obj.param]

Les champs automatiques spéciaux retournent des informations relatives au moteur de fusion.

Information Champ automatique spécial
Version de TinyButStrong : [onshow..version]
Nom du script PHP : [onshow..script_name]
Nom du modèle chargé : [onshow..template_name]
Répertoire du modèle chargé : [onshow..template_path]
Date de création du modèle chargé : [onshow..template_date;frm='dd/mm/yyyy hh:nn:ss']
Date et heure actuelle : [onshow..now;frm='dd/mm/yyyy hh:nn:ss']
