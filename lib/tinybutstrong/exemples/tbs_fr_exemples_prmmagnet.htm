Exemple de balises aimants

Une balise aimant est une balise Html attachée à un champ TBS. La balise aimant est supprimée quand le champ est supprimé, c'est à dire quand le champ a une valeur nulle ou chaîne vide. Une balise aimant est définie grâce au paramètre magnet. Le type de balise (simple ou ouvrant-fermant) et sa position sont définies grâce au paramètre mtype.


Valeurs non vides Valeurs vides
Paramètre Valeur Affichage
     
magnet=a $url = '[onshow.url]' Lien : Site TBS
     
magnet=a;mtype=m+m $url = '[onshow.url]' Lien : Site TBS
     
magnet=img;mtype=m* $image = '[onshow.image]' Image :
     
magnet=br; mtype=*m $line2 = '[onshow.line2]' Adresse :

[onshow.line1;magnet=br;mtype=*m]
[onshow.line2;magnet=br;mtype=*m]
