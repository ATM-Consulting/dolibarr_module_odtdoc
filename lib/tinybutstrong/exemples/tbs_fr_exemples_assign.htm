Exemple de données assignées

La propriété Assigned vous permet de définir les informations pour une fusion ultérieures qui peut être automatique (sur les évènements onload ou onshow) ou manuelle. La propriété Assigned est un tableau PHP qui contient les informations (paramètres et/ou données préparées).
