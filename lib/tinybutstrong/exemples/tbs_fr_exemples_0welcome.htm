Exemples de fusion avec TinyButStrong

Utilisation des exemples :
1/ Choisissez un exemple dans le menu à gauche.
2/ Changez l'affichage dans le menu à gauche pour regarder
- le résultat de la fusion.
- le contenu du modèle HTML,
- le source du modèle HTML,
- du programme PHP correspondant,
   
Installation des exemples :

Si ces exemples ne sont pas placés sous un environnement qui traite les scripts PHP, alors seuls les modèles HTML pourront être affichés. Le résultat de la fusion, le source du modèle HTML ainsi que le programme PHP ne seront pas accessibles.

-> Base de données MySQL :

Pour afficher les exemples qui utilisent des données MySQL, vous devez créer une table à laide du script du fichier tbs_fr_exemples_datamysql.sql,  puis modifier le fichier tbs_fr_exemples_datamysql.php pour réaliser la connexion à la base.

-> Autres bases de données :

Les exemples des autres bases de données ne s'affichent jamais en mode 'Résultat'.
