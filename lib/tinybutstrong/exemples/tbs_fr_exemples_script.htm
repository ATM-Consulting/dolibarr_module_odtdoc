Exemple d'exécution de script Php

Le paramètre script permet d'exécuter un script Php au moment de la fusion d'un champ. Si le script doit afficher du texte, vous pouvez lui indiquer de s'exécuter en tant que sous-modèle grâce au paramètre subtpl.

[onshow.test;script=[val];subtpl]
