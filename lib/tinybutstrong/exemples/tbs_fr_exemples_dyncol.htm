Exemple de colonnes dynamiques

La création de colonnes dynamiques avec TinyButStrong est simple mais un peu délicate. Il n' a pas besoin de paramètres spécifiques. Mais il faut penser à étendre les colonnes sur toutes les lignes du tableau de base en utilisant un bloc différent pour chaque ligne. La fusion des colonnes doit se faire avant la fusion des lignes.

Dans cet exemple, le script crée une table de multiplication avec alternance de couleur de ligne. Vous pouvez faire varier le nombre de colonnes et de lignes.
Nombre de lignes :       Nombre de colonnes :      
X
[c0.key;block=td]
[r.$;block=tr]
[r.[c1.val;block=td]]
[r.$;block=tr]
[r.[c2.val;block=td]]
