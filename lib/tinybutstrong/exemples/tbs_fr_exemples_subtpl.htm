Exemple de sous-modèle

Le paramètre file permet d'inclure un sous-modèle statique dans le modèle principal.
Le paramètre script, avec le paramètre subtpl, permettent d'inclure un sous-modèle dynamique.

Dans cet exemple, l'entête et le pied de page sont des sous-modèles statiques. Les articles sont aussi des sous-modèles statiques mais ils dépendent d'une variable globale ($tmpl_article). La zone de login est un sous-modèle dynamique. Elle est gérée par un sous-script qui peut être exécuté dans le mode Sou-modèle de TBS.


[onload;file='tbs_fr_exemples_subtpl_header.htm';getbody]
[onload;script=tbs_fr_exemples_subtpl_login.php;subtpl;getbody]
[onload;file=[var.tmpl_menu];getbody] [onload;file=[var.tmpl_article];getbody]
 
[onload;file='tbs_fr_exemples_subtpl_footer.htm';getbody]
