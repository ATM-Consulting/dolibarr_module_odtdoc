Exemple d'utilisation du paramètre serial

Le paramètre serial permet d'afficher plusieurs enregistrements dans un bloc principal. Pour cela, on utilise des sous-blocks.

Exemple d'utilisation de serial pour un affichage multi-colonnes :

Position / Titre
>
[bx_1.rank]
[bx_1.title;block=td]
[bx_2.rank]
[bx_2.title;block=td;.]
[bx_3.rank]
[bx_3.title;block=td;.]
[bx;block=tr;serial]
  Cette ligne est optionnelle, elle
premet de définir la case vide.

[bx_0;block=td]
[bx;block=tr;serial]


Exemple d'utilisation de serial pour un affichage complexe :

Position / Titre
>
[bz_1.rank] [bz_1.title;block=td]
[bz_3.rank] [bz_3.title;block=td;.]
 
[bz_2.rank] [bz_2.title;block=td;.]
 
[bz;block=tr;serial]
